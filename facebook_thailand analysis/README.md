
# 📊 Thailand Facebook Live Dataset — Exploratory Data Analysis (EDA)

## 📌 Overview

This project performs a comprehensive **Exploratory Data Analysis (EDA)** on the Thailand Facebook Live dataset. The dataset captures user engagement metrics (likes, comments, shares, reactions) for Facebook posts, enabling analysis of **content performance and audience behavior**.

The goal is to uncover patterns, relationships, and key drivers of engagement using statistical and visual analysis.

---

## 🎯 Objectives

* Understand the distribution of engagement metrics
* Identify which content types perform best
* Analyze relationships between reactions, comments, and shares
* Detect trends and anomalies in user interaction
* Provide actionable insights for social media strategy

---

## 📂 Dataset Description

The dataset consists of Facebook Live post data with features such as:

* **Status Type** (video, photo, link, status)
* **Number of Reactions**
* **Likes, Loves, Wows, Hahas, Sads, Angrys**
* **Comments Count**
* **Shares Count**
* **Post Timing Information**

---

## 🧱 Workflow

### 1. Data Understanding

* Inspected dataset structure
* Checked data types and basic statistics
* Identified key variables for analysis

### 2. Data Cleaning

* Handled missing values (if any)
* Removed redundant or irrelevant columns
* Ensured consistency in data types

### 3. Univariate Analysis

* Distribution of reactions, comments, and shares
* Identification of skewness and outliers
* Frequency distribution of post types

### 4. Bivariate & Multivariate Analysis

* Correlation between engagement metrics
* Relationship between post type and engagement
* Comparative analysis across different reactions

### 5. Visualization

* Histograms for distribution analysis
* Boxplots for outlier detection
* Heatmaps for correlation analysis
* Bar charts for categorical comparisons

---

## 📊 Key Insights

* **Highly skewed engagement metrics**
  → Few posts receive very high engagement while most remain low

* **Strong correlation between reactions, comments, and shares**
  → Viral posts tend to perform well across all metrics

* **Post type significantly impacts engagement**
  → Certain formats (e.g., videos) outperform others

* **Outliers represent viral content**
  → Important for understanding peak engagement behavior
