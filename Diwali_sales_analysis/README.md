
# 🪔 Diwali Sales Dataset

## 📌 Overview

The **Diwali Sales Dataset** contains transactional and demographic data of customers who made purchases during the Diwali festival season. It is widely used for practicing **Exploratory Data Analysis (EDA)**, customer segmentation, and deriving business insights related to retail and consumer behavior.

This dataset helps analyze how different demographic factors such as **age, gender, marital status, occupation, and location** influence purchasing patterns.

---

## 📂 Dataset Information

* **Domain:** Retail / E-commerce
* **Type:** Structured (Tabular Data)
* **Format:** CSV
* **Use Case:** EDA, Data Visualization, Business Insights, Customer Segmentation

---

## 📊 Features Description

| Column Name      | Description                                |
| ---------------- | ------------------------------------------ |
| User_ID          | Unique identifier for each customer        |
| Cust_name        | Name of the customer                       |
| Product_ID       | Unique product identifier                  |
| Gender           | Gender of the customer (Male/Female)       |
| Age Group        | Age category (e.g., 18–25, 26–35, etc.)    |
| Age              | Age of the customer                        |
| Marital_Status   | 0 = Unmarried, 1 = Married                 |
| State            | State of residence                         |
| Zone             | Region classification (North, South, etc.) |
| Occupation       | Customer profession                        |
| Product_Category | Category of product purchased              |
| Orders           | Number of orders placed                    |
| Amount           | Total purchase amount                      |

---

## 🎯 Objective of the Dataset

The dataset is designed to:

* Understand **customer buying behavior during festive seasons**
* Identify **high-value customer segments**
* Analyze **regional sales trends**
* Study **product demand patterns**
* Support **data-driven marketing strategies**
