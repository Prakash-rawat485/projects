
🚗 Used Car EDA
📌 Overview

This project performs Exploratory Data Analysis (EDA) on a used car dataset to understand price drivers, depreciation patterns, and market trends. The goal is to extract insights useful for buyers, sellers, and dealerships.

🎯 Key Objectives
Identify factors affecting resale price
Analyze impact of age, kms driven, fuel type, transmission
Discover market patterns and pricing behavior
🛠️ Tech Stack
Python
Pandas, NumPy
Matplotlib, Seaborn
Jupyter Notebook
🔍 Analysis Performed
Data Cleaning: handled missing values, duplicates
Univariate: price distribution, fuel type frequency
Bivariate: price vs year, kms driven, fuel type
Multivariate: correlation analysis & feature interactions
📊 Key Insights
📉 Price decreases with car age and kms driven
⛽ Diesel cars retain better resale value
⚙️ Automatic cars are priced higher than manual
👤 Fewer owners → higher resale price
🏢 Dealers list cars at higher prices than individuals
